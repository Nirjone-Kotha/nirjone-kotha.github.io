# V5.3.4 focused fixes

Only the two requested areas were changed.

1. Primary destinations such as Calm space, Profile/Edit profile, Support circles, Circle details, Contact, Cookie settings and Safety now open as full direct pages with a visible Back button instead of centered popup cards. Action-oriented interfaces such as post composer, comments, story composer, filters and install prompts remain contextual overlays.
2. A normal phone browser now uses the same compact layout as the installed phone app. When the user explicitly enables Desktop site in the phone browser, the complete desktop layout remains active. Tablet, laptop and desktop behaviour is unchanged.

Service-worker and asset version: 5.3.4.
