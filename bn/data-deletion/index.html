ডেটা মুছে ফেলা

Profile and settings থেকে app data export বা reset করা যায়। Reset করলে local data এবং signed in থাকলে password profile-এর synced app data clear হয়।

অ্যাপ খুলুননিরাপদ ব্যবহার
এটি চালুর আগের সম্পাদনাযোগ্য খসড়া; চূড়ান্ত আইনগত পর্যালোচনা প্রয়োজন।

Guest profile reset করুন

Profile and settings খুলে Reset local data নির্বাচন করে নিশ্চিত করুন। Browser settings থেকেও site data clear করা যায়। Guest data clear হলে ফিরিয়ে আনা যাবে না।

Password profile reset করুন

Signed in অবস্থায় একই reset control প্রথমে Firebase UID-এর অধীনে synced appData সরায়, তারপর এই device-এর data clear করে। একই password দিয়ে ভবিষ্যতে ব্যবহারের জন্য pseudonymous Authentication identity থেকে যায়।

Installed copy-এর আলাদা storage হতে পারে

আলাদাভাবে install করা PWA-এর নিজস্ব local storage থাকতে পারে। Guest mode ব্যবহার করলে প্রতিটি installation-এর data আলাদাভাবে clear করুন।

সাপোর্টে যোগাযোগ

যাচাইকৃত support detail config-এ যোগ হওয়ার পর contact page privacy question-এর মাধ্যম দেবে।

সাধারণ প্রশ্ন

Uninstall করলেই কি সব data মুছে যায়?

সবসময় নয়। In-app reset ও browser site-data control ব্যবহার করুন।

Clear করা Guest data কি ফিরবে?

না। Guest mode-এ cloud backup নেই।

Reset করলে password identity-ও delete হবে?

না। Synced app data clear হয়; pseudonymous Firebase Authentication identity থেকে যায়।
