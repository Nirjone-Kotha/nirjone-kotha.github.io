কুকি ও লোকাল স্টোরেজ নীতি

প্রয়োজনীয় local storage interface preference ও Guest activity device-এ রাখে। ঐচ্ছিক password profile restore করতে Firebase Authentication ও Realtime Database ব্যবহার হয়।

অ্যাপ খুলুননিরাপদ ব্যবহার
এটি চালুর আগের সম্পাদনাযোগ্য খসড়া; চূড়ান্ত আইনগত পর্যালোচনা প্রয়োজন।

প্রয়োজনীয় local storage

ভাষা, theme, consent choice ও বর্তমান app state localভাবে থাকে। Guest post ও comment শুধু সেই browser-এ থাকে।

ঐচ্ছিক password profile storage

Password profile নিলে app data restore করার জন্য Firebase Authentication ও Realtime Database ব্যবহার হয়। Visitor বাস্তব email address বা phone number দেয় না।

ঐচ্ছিক analytics

কোনো analytics provider কনফিগার করা নেই। ভবিষ্যতে যোগ হলে সম্মতির অপেক্ষা করবে এবং আবেগের লেখা, মুড, সার্চ বা safety detail নেবে না।

ঐচ্ছিক বিজ্ঞাপন

বিজ্ঞাপন বন্ধ। ভবিষ্যতের non-personalized ad-এ consent প্রয়োজন এবং সংবেদনশীল অংশে বিজ্ঞাপন থাকবে না।

পছন্দ পরিবর্তন

অ্যাপে cookie settings রয়েছে, যাতে পরে ঐচ্ছিক category আবার পরিবর্তন করা যায়।

সাধারণ প্রশ্ন

না। analytics ও advertising consent defaultভাবে বন্ধ।
