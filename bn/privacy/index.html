গোপনীয়তা নোটিশ

মনের কথায় বাস্তব email বা phone number না দিয়েই ঐচ্ছিক password-only profile ব্যবহার করা যায়। চাইলে Guest হিসেবে device-only storage-এ চালিয়ে যাওয়া যায়।

অ্যাপ খুলুননিরাপদ ব্যবহার
এটি চালুর আগের সম্পাদনাযোগ্য খসড়া; চূড়ান্ত আইনগত পর্যালোচনা প্রয়োজন।

Password-only Firebase profile

Password profile নিলে app একটি pseudonymous Firebase Authentication account তৈরি করে এবং আপনার app data private Firebase UID-এর অধীনে রাখে। বাস্তব email address বা phone number দিতে হয় না।

Guest mode ও local storage

Password option Skip করলে post, comment, preference ও check-in বর্তমান browser-এ থাকে। Site data মুছে গেলে বা device বদলালে Guest profile ফিরিয়ে আনা যাবে না।

Password-এর দায়িত্ব

Password-টিই profile ID ও access key। Recovery channel নেই। একই password জানে এবং ব্যবহার করে এমন কেউ একই profile খুলতে পারে, তাই শক্তিশালী ও unique passphrase ব্যবহার করুন এবং অন্য কোথাও reuse করবেন না।

আবেগসংক্রান্ত তথ্য বিজ্ঞাপনের সংকেত নয়

মুড, প্রয়োজন, post text, comment, search, saved content ও safety interaction বিজ্ঞাপন targeting বা remarketing-এ ব্যবহার করা যাবে না।

আপনার নিয়ন্ত্রণ

Profile and settings থেকে app data export বা reset করা যায়। Password profile-এ signed in থাকলে reset-এর সময় সংশ্লিষ্ট synced app data-ও clear হয়।

সাধারণ প্রশ্ন

Phone number বা বাস্তব email দিতে হবে?

না। Visitor profile শুধু password নেয় এবং pseudonymous Firebase identity ব্যবহার করে।

Password ভুলে গেলে recovery হবে?

না। Phone বা email recovery channel নেই, তাই password মনে রাখতে হবে।

মুড কি বিজ্ঞাপনে ব্যবহার হয়?

না। বিজ্ঞাপন স্থাপত্যে এটি স্পষ্টভাবে নিষিদ্ধ।

সম্পর্কিত পেজ

কুকি ও লোকাল স্টোরেজ নীতিডেটা মুছে ফেলানিরাপত্তা তথ্যযোগাযোগ

মনে রাখুন

এটি সহমর্মিতাভিত্তিক সামাজিক সহায়তার জায়গা; চিকিৎসা, রোগ নির্ণয় বা জরুরি সেবা নয়।
