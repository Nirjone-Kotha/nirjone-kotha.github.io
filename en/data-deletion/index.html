Data Deletion

Profile and settings can export or reset app data. Reset clears local data and, while signed in, the synced app data for that password profile.

Open appUse safely
Editable pre-launch draft: owner and legal review are still required.

Reset a guest profile

Open Profile and settings, choose Reset local data, and confirm. You can also clear site data from browser settings. Guest data cannot be restored after it is cleared.

Reset a password profile

While signed in, the same reset control first removes the synced appData under your Firebase UID and then clears this device. The pseudonymous Authentication account itself remains available for future use with the same password.

Installed copies may have separate storage

A separately installed PWA can have its own local storage. Clear each installation if it was used in Guest mode.

Contact support

Once verified support details are configured, the contact page will provide a channel for privacy questions.

Frequently asked questions

Does uninstalling always clear data?

Not necessarily. Use the in-app reset and browser site-data controls.

Can cleared guest data be restored?

No. Guest mode has no cloud backup.

Does reset delete the password identity itself?

No. It clears synced app data; the pseudonymous Firebase Authentication identity remains.
